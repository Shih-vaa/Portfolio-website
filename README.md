# 🌐 Apoorv Pandey — Portfolio Website

Welcome to my **portfolio website** — a reflection of my work, skills, and journey as a **freelance developer**.  
Built with **React**, **Vite**, and **Tailwind CSS**, this portfolio showcases who I am, what I’ve built, and how to connect with me.



---

## 🧭 Overview

My portfolio includes:

- **Home** – A short introduction and summary.  
- **Skills** – Technologies I specialize in.  
- **Experience** – My professional background.  
- **Projects** – Real-world projects I’ve built and delivered.  
- **Education** – My academic background.  
- **Contact** – Easy ways to reach me.  

---

## ⚙️ Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Icons:** React Icons  
- **Deployment:** Vercel
